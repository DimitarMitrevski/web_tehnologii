Вежба со  SASS.

1. Во htdocs папката на XAMPP (WAMP, LAMP...) се креира веб страница index.html и две папки: css и sass

2. Веб страницата index.html се поврзува со styles.css стандардно со link елемент.

3. Во css папката се креира датотека styles.css и се остава празна.

4. Во sass папката се креира датотека styles.scss и во неа ќе се уредува дизајнот на веб страницата со SASS.

5. Се симнува апликација за компајлирање на .scss датотеките во .css. Во оваа вежба се користи Scout App од https://scout-app.io/

6. Се стартова апликацијата Scout App. На првиот прозорец се избира + Add a Project.
Потоа се избира htdocs папката од XAMPP серверот. На следниот прозорец има испишани
патеки на два Stylesheet директориуми: Input Folder и Output Folder. Како Input Folder е scss папката,
а како Output Folder е css папката. Ако случајно не се соодветно избрани папките, се избираат рачно.
Со тоа подесувањето е завршено.

7. Следниот чекор е да се кликне на копчето "play" кај ознаката на htdocs папката од прозорецот на Scout App.

Во "play" мод секоја снимена промена во styles.scss Scout App автоматски ја снима во styles.css. 

Да се изработат повеќе вежби од https://sass-lang.com/guide со различни HTML елементи!
